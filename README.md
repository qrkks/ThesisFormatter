# format_macro.bas - Word 文档格式化宏

这是一个用于自动格式化 Word 文档的 VBA 宏集合，特别适用于学术论文、报告等文档的格式化需求。该宏文件提供了完整的文档格式化功能，包括标题、正文、摘要、目录、参考文献和图片的自动格式化。

## 功能概述

该宏文件提供了以下主要功能：

### 1. 标题格式化

- **题目格式化** (`FormatTitleByHeadingStyle`): 格式化文档标题（"标题"样式）
- **一级标题** (`FormatLevel1Heading`): 格式化一级标题（居中，黑体，小三）
- **二级标题** (`FormatLevel2Heading`): 格式化二级标题（左对齐，黑体，四号）
- **三级标题** (`FormatLevel3Heading`): 格式化三级标题（左对齐，黑体，小四）

### 2. 正文格式化

- **正文格式化** (`FormatBodyText`): 设置正文格式（宋体，小四，首行缩进两字符）
- **Compact 样式** (`FormatCompactParagraph`): 无缩进的正文格式

### 3. 页面设置

- **页面和行距设置** (`SetPageAndBodyFormat`):
  - A4 纸张
  - 页边距：上下 3cm/2.5cm，左右 3cm/2.5cm
  - 行距：1.5 倍行距

### 4. 摘要和关键词

- **摘要格式化** (`MergeAndFormatAbstract`):
  - 自动合并"摘要"、"关键词"、"Abstract"、"Keywords"及其内容
  - 自动添加冒号（如需要）
  - 标题加粗，内容正常字体
  - 支持中英文摘要

### 5. 目录处理

- **查找目录位置** (`FindTableOfContentsPosition`): 定位文档中的"目录"标记
- **插入目录** (`InsertTableOfContents`): 自动插入 Word 目录（幂等实现）
- **更新目录** (`UpdateTableOfContents`): 更新现有目录
- **格式化目录条目** (`FormatTableOfContentsEntries`): 格式化目录样式
- **完整目录处理** (`ProcessTableOfContents`): 一键处理目录（插入/更新+格式化）

### 6. 参考文献处理

- **参考文献标题格式化** (`FormatReferences`): 格式化"参考文献"或"References"标题，并添加分页符
- **参考文献条目格式化** (`FormatReferenceEntries`): 格式化参考文献条目（悬挂缩进，APA 标准）
- **自动编号** (`AutoNumberReferences`): 为参考文献条目自动添加编号
- **参考文献排序** (`SortReferences`): 按字母顺序排序参考文献
- **完整参考文献处理** (`ProcessReferencesWithSort`): 一键处理参考文献（格式化+编号+排序）

### 7. 图片处理

- **图片居中** (`FormatImages`): 自动将文档中的所有图片居中显示
- **图片标题格式化** (`FormatImageCaptions`): 格式化图片标题（"图 X" 或 "Figure X"）
- **完整图片处理** (`ProcessImages`): 一键处理图片（居中+标题格式化）

### 8. 一键格式化

- **山东中医药大学论文格式化** (`FormatThesisToSDUTCM`): 一键执行山东中医药大学论文格式化流程（带确认对话框）

## 使用方法

### 安装宏

1. 打开 Word 文档
2. 按 `Alt + F11` 打开 VBA 编辑器
3. 在左侧项目资源管理器中，找到你的文档或模板
4. 右键点击 `Microsoft Word 对象` → `插入` → `模块`
5. 将 `format_macro.bas` 文件的内容复制到新模块中
6. 保存文档（建议保存为 `.docm` 格式以支持宏）

### 运行宏

#### 方法一：使用 VBA 编辑器

1. 按 `Alt + F11` 打开 VBA 编辑器
2. 在代码窗口中找到要运行的宏（如 `FormatThesisToSDUTCM`）
3. 将光标放在宏内部，按 `F5` 运行

#### 方法二：使用宏对话框

1. 按 `Alt + F8` 打开宏对话框
2. 选择要运行的宏（如 `FormatThesisToSDUTCM`）
3. 点击"运行"

#### 方法三：添加到快速访问工具栏

1. 文件 → 选项 → 快速访问工具栏
2. 从"从下列位置选择命令"下拉菜单中选择"宏"
3. 选择要添加的宏，点击"添加"
4. 点击"确定"，即可在快速访问工具栏中快速运行宏

### 推荐使用流程

对于新文档，推荐使用一键格式化：

```vba
FormatThesisToSDUTCM
```

该宏会按顺序执行以下步骤：

1. 页面设置和正文行距
2. 标题格式化（题目、一级、二级、三级标题）
3. 正文格式化
4. 摘要和关键词格式化
5. 目录处理
6. 参考文献格式化（包含排序）
7. 图片居中处理

## 格式化规范

### 字体规范

- **中文**: 宋体
- **英文/数字**: Times New Roman（标题和正文）
- **标题**: 黑体（中文标题）

### 字号规范

- **题目**: 小二（18pt）
- **一级标题**: 小三（16pt）
- **二级标题**: 四号（14pt）
- **三级标题**: 小四（12pt）
- **正文**: 小四（12pt）

### 段落格式

- **题目**: 居中
- **一级标题**: 居中
- **二级标题**: 左对齐
- **三级标题**: 左对齐
- **正文**: 左对齐，首行缩进两字符（24pt）
- **参考文献**: 悬挂缩进（APA 标准）

### 行距

- 全文：1.5 倍行距

### 页边距

- 上：3cm
- 下：2.5cm
- 左：3cm
- 右：2.5cm

## 注意事项

1. **文档样式要求**: 宏依赖于 Word 的样式系统，确保文档使用了正确的样式：
   - 标题使用"标题"样式
   - 一级标题使用"Heading 1"或"标题 1"样式
   - 二级标题使用"Heading 2"或"标题 2"样式
   - 三级标题使用"Heading 3"或"标题 3"样式
   - 正文使用"正文文本"、"Normal"、"First Paragraph"或"正文"样式

2. **摘要和关键词格式**: 宏会自动识别以下格式：
   - "摘要"、"关键词"（中文）
   - "Abstract"、"Keywords"（英文）
   - 支持带冒号和不带冒号的格式

3. **目录处理**:
   - 宏会自动在"目录"标记处插入或更新目录
   - 如果文档中没有"目录"标记，目录处理会被跳过

4. **参考文献处理**:
   - 参考文献部分会自动添加分页符
   - 参考文献条目会自动应用悬挂缩进（APA 标准）
   - 支持自动编号和排序功能

5. **性能优化**:
   - 使用 `FormatThesisToSDUTCM` 可以获得更好的性能
   - 格式化过程中会关闭屏幕更新以提高速度

6. **错误处理**:
   - 宏包含基本的错误处理机制
   - 如果遇到错误，会显示错误消息并恢复屏幕更新

## 自定义修改

如果需要修改格式化规范（如字体、字号、页边距等），可以直接编辑相应的宏函数。主要修改点：

- **字体和字号**: 修改各格式化函数中的 `.Font.Name`、`.Font.NameFarEast` 和 `.Font.Size` 属性
- **页边距**: 修改 `SetPageAndBodyFormat` 中的 `TopMargin`、`BottomMargin`、`LeftMargin`、`RightMargin` 值
- **行距**: 修改 `SetPageAndBodyFormat` 中的 `LineSpacingRule` 属性
- **缩进**: 修改各格式化函数中的 `FirstLineIndent` 和 `LeftIndent` 值

## 兼容性

- **Word 版本**: 适用于 Microsoft Word 2010 及更高版本
- **操作系统**: Windows（VBA 宏仅在 Windows 版 Word 中可用）
- **文档格式**: 支持 `.docx` 和 `.docm` 格式（建议使用 `.docm` 以保存宏）

## 常见问题

**Q: 运行宏后没有效果？**  
A: 检查文档是否使用了正确的样式。宏依赖于 Word 样式系统，如果段落没有应用相应的样式，格式化不会生效。

**Q: 目录没有自动插入？**  
A: 确保文档中有"目录"文本标记。如果没有，宏会跳过目录处理。

**Q: 参考文献格式不正确？**  
A: 确保参考文献部分以"参考文献"或"References"开头，且每个参考文献条目是独立的段落。

**Q: 图片没有居中？**  
A: 检查图片的环绕方式。某些特殊的图片格式可能需要手动调整。
